Show Your Facebook Photos on the Website

Your website can automatically display the photos you post on your Facebook Page — styled to match your Gallery. The code is already built and waiting. It just needs one thing that only you can create: a Facebook access token. This guide walks you through it. No technical knowledge needed — you do the clicking, and we handle everything technical.

First, an honest note about Facebook reviews Facebook photos can appear automatically (this guide). Facebook reviews cannot — Meta shut down public access to Page reviews, so no website can pull them. The reliable way to show your best Facebook recommendations is to feature them manually as testimonials on the site. This guide is about photos.

iHow this works

You do the parts that need your Facebook login. Then you send us three values, and we do all the technical work — you never touch the hard part.

✋ You do (needs your account)

  • Create a Facebook App (~3 min)
  • Grant two read-only permissions
  • Copy 3 values and send them to us privately

🛠 We do (the technical part)

  • Exchange your values into a long-lived, non-expiring Page token
  • Find your Page's photo access automatically
  • Install it and confirm your real photos appear in the Gallery
Good news: it won't keep expiring Done the proper (long-lived) way we handle for you, the token is effectively permanent. It only breaks if you change your Facebook password or remove the app.

1Create a Facebook App (~3 min)

  1. Go to developers.facebook.com → top-right My Apps → Create App.
  2. If asked "What do you want your app to do?", pick Other → Next.
  3. App type: Business → Next.
  4. Name it Emcey Brows Website, add your email → Create app (re-enter your Facebook password if prompted).

2Open Graph API Explorer & grant permissions

  1. In the left menu (or search) open Tools → Graph API Explorer — or go to developers.facebook.com/tools/explorer.
  2. Top-right Meta App dropdown → select Emcey Brows Website.
  3. Click Add a Permission and tick these two: pages_show_list and pages_read_engagement. (Both are read-only — they let us read your photos, nothing else.)
  4. Click Generate Access Token → a popup asks you to log in and allow access to your Emcey Brows page → approve it.
Why no "app review" is needed Because it's your page and your app, Facebook lets you read your own photos immediately — you skip the weeks-long approval process that trips most people up.

3Send us three values (privately)

Copy these and send them to us

  1. App ID — from your app's Settings → Basic page.
  2. App Secret — same page, click Show to reveal it.
  3. Access token — the long string now showing in the Graph API Explorer box.
ValueWhere to find it
App IDApp dashboard → Settings → Basic
App SecretApp dashboard → Settings → Basic → Show
Access tokenTools → Graph API Explorer (the token box)

Then we finish it for you

  • Exchange your values into a long-lived, non-expiring Page token
  • Detect your Page's photo access automatically
  • Install it and load your Gallery to confirm your real Facebook photos appear
Keep these private — and reset afterwards The App ID, App Secret, and token are sensitive. Send them privately, never post them publicly. Once everything works, it's good practice to reset the App Secret (Settings → Basic) — the long-lived Page token we generate keeps working regardless.
