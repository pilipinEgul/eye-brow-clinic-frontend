Setting Up Google & Facebook Reviews

The one thing you must do To turn on automatic Google reviews, create a Google API key and paste it into the site's settings — that's it. Steps are in Section 2.

1Google reviews — the quick link-out (already done)

On the Testimonials page there are buttons that send visitors straight to your Google listing:

  • ★ Read Google reviews — opens your Google Maps listing
  • Write a Google review — opens Google's review box

These already point at the Emcey Brows listing — nothing to do. If your business listing ever changes, update the Place ID in the site settings (file lib/site.ts, value placeId).

How to find your Google Place ID (only if it changes)

  1. Go to Google's Place ID finder: developers.google.com/maps/.../place-id
  2. Search “Emcey Brows Aesthetics – Imus” on the map.
  3. Copy the Place ID shown in the bubble (looks like ChIJ_x9hcQDTlzMR5JsBByxQ79o).

2Turn ON automatic Google reviews

This makes real Google reviews appear on the site and refresh themselves. It is already coded — it only needs a Google API key. Until a key is added, the page safely shows the link-out buttons instead.

Step A — Create the Google key (one time, ~10 min)

  1. Open the Google Cloud Console: console.cloud.google.com and sign in with the clinic's Google account.
  2. Create a project (top bar → New Project → name it “Emcey Brows Website”).
  3. In the search bar type “Places API (New)”, open it, and click Enable.
  4. Enable Billing on the project (a card is required even though normal website use stays within the free monthly allowance).
  5. Go to APIs & Services → Credentials → Create credentials → API key. Copy the key (starts with AIza…).
  6. Click Edit on the key → API restrictions → restrict it to Places API (New) → Save.

Step B — Add the key to the website

Open the site's .env file (or your hosting provider's Environment Variables settings) and add this line:

GOOGLE_PLACES_API_KEY=AIza...your-key-here...
Keep this key private It has no NEXT_PUBLIC_ prefix on purpose — that keeps it on the server only, so visitors never see it. Don't paste it into public files or chat.

Then restart / redeploy the site. Live Google reviews now appear automatically on the Testimonials page.

To turn it OFF

Delete or blank the GOOGLE_PLACES_API_KEY line and redeploy. The page falls back to the link-out buttons. No code change needed.

Good to know

  • Google returns up to 5 reviews and chooses which five — you can't display all of them or pick favourites. To feature a specific review, add it as a manual testimonial (Section 4).
  • Reviews are shown unmodified with the Google logo and reviewer name — this is required by Google and already handled.
  • The site caches reviews for 24 hours to stay within Google's free usage and refresh rules.

3Facebook reviews

Facebook calls reviews “Recommendations.” The website already has buttons that open them:

  • Read Facebook reviews — opens your Page's Reviews tab
  • Recommend us on Facebook — opens your Page

Make sure your buttons point to the right Page

  1. Open your Facebook Page (not a personal profile).
  2. Make sure Reviews/Recommendations are turned on: Page → Settings → Privacy → Page and tagging (or Templates and tabs → enable the Reviews tab).
  3. Confirm these two values in the site settings (file .env) match your real Page:
NEXT_PUBLIC_FACEBOOK_URL=https://www.facebook.com/profile.php?id=YOUR_PAGE_ID
NEXT_PUBLIC_FACEBOOK_REVIEWS_URL=https://www.facebook.com/profile.php?id=YOUR_PAGE_ID&sk=reviews
Why Facebook reviews can't auto-update like Google Facebook (Meta) shut down public, automatic access to Page reviews. Pulling them now would require a Facebook App, Business Verification, a special “Page Public Content Access” permission with app review, and tokens that expire — fragile and routinely rejected. The reliable approach is the link-out button here, plus manually featuring your best Facebook recommendations as testimonials (Section 4).

4Featuring specific reviews (manual)

For full control — to spotlight your very best Google or Facebook reviews in the homepage slider and testimonials grid — copy them into the admin as testimonials. This is the recommended, fully compliant way to highlight chosen reviews.

  1. When a client leaves a great review on Google or Facebook, copy the text.
  2. In the website admin, add a new Testimonial: name, star rating, the quote, and (optionally) which service it's about.
  3. Save — it appears on the homepage and Testimonials page right away.
