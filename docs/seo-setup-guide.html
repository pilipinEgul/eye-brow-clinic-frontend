iThe big picture

SEO is how your website earns free traffic from Google. For a local clinic, two things matter most:

  • Your Google Business Profile (the Maps listing) — this drives most "near me" and map results.
  • The website's on-page SEO — titles, descriptions, sitemap, and structured data. This is already built in.
What you actually have to do The technical SEO is done. Your job is mostly account setup: verify the site in Google Search Console, submit the sitemap, and keep your Google Business Profile accurate. Steps below.

One critical setting before launch The website's address must be set to your real domain. In the site's .env file set: 
NEXT_PUBLIC_SITE_URL=https://www.emceybrows.com
This one value feeds the sitemap, links, and all the schema. If it's left as a temporary/ngrok address, Google indexes the wrong URLs.

2Google Search Console (verify + submit sitemap)

Search Console is Google's free dashboard showing how your site performs in search. Set it up once.

  1. Go to search.google.com/search-console and sign in with the clinic's Google account.
  2. Click Add property → choose URL prefix → enter your full site address (e.g. https://www.emceybrows.com).
  3. Choose the HTML tag verification method. Copy the code — it looks like <meta name="google-site-verification" content="abc123…">. You only need the content value (abc123…).
  4. Give that value to your developer / paste it into the site's .env file: 
    GOOGLE_SITE_VERIFICATION=abc123...
    The website adds the verification tag automatically. Redeploy, then click Verify in Search Console.
  5. Once verified: open Sitemaps in the left menu, enter sitemap.xml, and click Submit.
Tip Prefer not to redeploy? You can instead verify by DNS record or by uploading the HTML file Google offers — any one method is enough.

3Google Business Profile (most important for local)

For a clinic, the Maps listing usually brings in more customers than the website itself. Claim and polish it.

  1. Go to business.google.com and search for “Emcey Brows Aesthetics – Imus.” Claim it (or create it if missing).
  2. Complete every field: category (e.g. “Beauty salon” / “Eyebrow bar”), address, hours, phone, website link, and services.
  3. Add lots of photos (studio, work, team) — listings with photos get far more clicks.
  4. Keep the NAP — Name, Address, Phone — written identically to the website. Consistency boosts local ranking.
  5. Ask happy clients for Google reviews (your website already has a "Write a Google review" button). Reviews strongly affect local ranking.

4Bing & analytics (optional)

  • Bing Webmaster Tools — bing.com/webmasters. You can import straight from Search Console, or verify with a code placed in .env as BING_SITE_VERIFICATION=….
  • Google Analytics 4 — to see visitor numbers. This needs a small developer add-on (not yet installed). Ask your developer to add GA4 if you want traffic stats.

6How to check it's working

  1. Rich Results Test — search.google.com/test/rich-results. Paste a page URL; it should detect your Beauty Salon and FAQ info.
  2. Visit https://your-domain/sitemap.xml and /robots.txt — they should show your real domain, not a test address.
  3. In Search Console → Pages, watch the number of indexed pages grow over the following days/weeks.
  4. Paste a page link into the Facebook Sharing Debugger to confirm the preview image and title (after the OG image is added).
SEO takes time Indexing and ranking improvements usually appear over weeks, not hours. Keep the Business Profile active and collect reviews steadily.
